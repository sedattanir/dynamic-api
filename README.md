# Dynamic Api Application

## Features
 - Express v4 Routes, middlewares and a lot more
 - Some security apps
 - Most of all DYNAMIC

## Installation

Firstly, Install dependencies for application

```sh
$ npm install
```

and then create .env files and run this command

```sh
$ npm start
```
